# Лабораторная работа №3 — Core Javascript

- Цель: реализовать набор функций на JavaScript по заданию из каркаса проекта.
- Каркас: https://github.com/teach-kirshdv/js-tasks (загружается в папку `lab3`).
- План выполнения:
  1) Создать ветку `lab3`.
  2) Дополнить README описанием ЛР3.
  3) Создать папку `lab3` и поместить туда каркас проекта `js-tasks`.
  4) Изучить задания и реализовать требуемые функции.
  5) Закоммитить и запушить изменения в ветку `lab3`.

## Как пользоваться

Для того, чтобы всё корректно работало, у вас должен быть установлен [node](https://nodejs.org/en/) и [git](https://git-scm.com/)

- Скачайте проект к себе. Есть 3 способа:
  - Скачайте проект как архив.
  - Склонируйте текущий репозиторий.
  - Сделайте fork репозитория и склонируйте его к себе.
- Скопируйте содержимое проекта в свой репозиторий по курсу.
- Разберитесь в структуре проекта.
- Задания находятся в `src/core.js`.
- Перед тем, как начать делать задание, выполните `npm install`.
